---How To Install On Linux---

cd Desktop

git clone https://github.com/ToxicOmaga/Omega-DDos-Tool

cd Omega-DDos-Tool

---How To Install On Termux---

pkg install python2

git clone https://github.com/ToxicOmaga/Omega-DDos-Tool

cd Omega-DDos-Tool

python2 Omega.py
